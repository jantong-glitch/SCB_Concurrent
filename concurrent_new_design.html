一併投保呈交的客戶聲明Customer Declaration on Concurrent Submission

此部分僅適用於客戶於同日簽署多份保險申請的情況。完成本申請後,客戶會否於同日簽署下一份保險申請? This section applies only where the client signs multiple insurance applications on the same day. After completing this insurance application, will the client sign the next insurance application on the same day?

一併投保呈交的定義:客戶於同日向同一銷售人員購買多份保單(不論產品相同或不同)。一併投保呈交只計算基本計劃。 Concurrent Submission is defined as: when a customer purchases multiple policies on the same day from the same sales staff (regardless of same or different products). Concurrent submission only counts for basic plan.

重要提示:Important Reminder: 請確保已選取所有適用的一併投保申請,否則同日簽署的所有保險申請均需重做。 Please ensure that all applicable concurrent applications are selected. Otherwise, all insurance applications signed on the same day will need to be reworked.
